Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Данная программа выведет: 77, 78, 79, так как мы передаем в переменную b наш массив а, и получаем его элементы с индексами от 1 до 4-х

```
